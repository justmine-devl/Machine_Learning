# Weights folder

Place model weights locally. Do not commit large model files to GitHub.

For OpenVINO ensemble inference:

```text
weights/openvino/model_0.xml
weights/openvino/model_0.bin
...
weights/openvino/model2_0.xml
weights/openvino/model2_0.bin
...
```

For PyTorch training or inference:

```text
weights/model_fold0.pth
```
